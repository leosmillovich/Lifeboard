## Setup

1. Copiar `.env.example` a `.env` y completar `VITE_SUPABASE_URL` y `VITE_SUPABASE_ANON_KEY`.
2. Ejecutar el SQL de `supabase/schema.sql` en tu proyecto de Supabase.
3. Instalar dependencias:

```bash
npm install
```

4. Correr en desarrollo:

```bash
npm run dev
```
